# Практика в SM lab

Проект создан на Vue3 и Vite

## Запустить проект

```sh
npm install
```

### Скомпилировать для hot-reload разработки

```sh
npm run dev
```

### Запустить Storybook

```sh
npm run storybook
```

### Запустить тесты

```sh
npm run test
```

### Запустить eslint

```sh
npm run lint
```

### Скомпилировать для production

```sh
npm run build
```
